Лабораторна робота №4 — Основи PHP

Що зроблено:
1) napr.php
   - читає napr.txt
   - перетворює кодування Windows-1251 -> UTF-8 (за потреби)
   - сортує напрями та формує HTML-сторінку з формою (radio + submit)

2) stat.php
   - приймає GET-параметр napr
   - шукає відповідний розділ у data.txt
   - зчитує N ВНЗ (N береться з наступного рядка після назви напряму)
   - виводить дані у таблицю
   - якщо «контракт» від’ємний — це недобір (підсвічено жирним)

Як запустити на хостингу/локально:
- Скопіюйте в одну папку файли:
  napr.php, stat.php, napr.txt, data.txt
- Відкрийте у браузері napr.php
- Додайте посилання з головної сторінки сайту на napr.php

Примітка про кодування:
- У вихідних файлах napr.txt / data.txt зазвичай використовується Windows-1251,
  тому в коді є iconv(...). Якщо ваші файли вже UTF-8 — все одно працюватиме.
